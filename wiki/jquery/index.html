    # jQuery

    Released on 2006 by John Resig. Designed to simplify HTML DOM tree traversal and manipulation, as well as event handling, CSS animation, and Ajax. At this time, there was no chrome and IE ruled browser. Lots of compatibility issues were solved by jQuery. Now that browsers handle JS the same way jQuery is no longer needed.

    Is jQuery dead? Many developers noted that they bypassed jQuery. Mainly because they started using React. This may be because of the virtual DOM (opens new window).
